Cloud Portfolio Website 

SOURCE CODE

A website, hosted on AWS cloud, for my portfolio. The interesting bit was to deploy everything using IaC and implementing CI/CD.

  • AWS SAM as the IaC solution
  • S3 to host file
  • Cloudfront and ACM to serve HTTPS requests
  • Route 53 for the DNS
  • As that point the website was running but I also added :
  • Lambda to build a function that count the number of visits on my website and returns the count
  • DynamoDB to store this data
  • API Gateway
  • to be able to display the data on our website by communicating with my function and my DB
  • Git as our VCS, with Github
  • Test with pytest to test our function
  • Github Actions to automate testing and deployment
